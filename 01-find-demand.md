# 01 - 找需求

> 不是从零想idea，而是找"已经有人做了"的产品，然后山寨。

## 🎯 核心策略

**看别人在做什么，是否受欢迎，然后山寨。**

### 为什么这样做？

```
✅ 需求已被验证（有人做了说明有需求）
✅ 看得到产品形态（不用瞎想）
✅ 可以快速评估（有数据信号）
✅ 降低失败率（比从零想idea靠谱）

❌ 不要：问用户需求（伪需求）
❌ 不要：自己想"革命性"idea（99%失败）
❌ 不要：追热点概念（AI、Web3等空泛方向）
```

## 📊 数据源

### 主要来源（已自动化）

**你的trend-monitor自动采集：**

1. **Product Hunt**
   - 每日trending产品

2. **There's An AI For That**

   - 分类清晰

3. **Chrome Extensions**
   - 用户数量
   - 评分
   - 增长趋势

4. **GitHub Trending**
   - 开源项目
   - Stars数量

5. **Hacker News**
   - Show HN帖子
   - 讨论热度

### 次要来源（手动查看）

- Reddit r/SideProject
- IndieHackers首页
- Twitter #buildinpublic

## 🔍 筛选标准

### 必须满足的条件

```
✅ Solo maker做的（1-2人团队）
   - 看产品页面的team介绍
   - 或者创始人的Twitter

✅ 有付费信号
   - 有pricing页面
   - 或者Chrome扩展是paid
   - 或者有明确的商业模式

✅ 技术上可实现
   - 1周能做出80%功能的MVP
   - 不需要复杂的AI训练
   - 不需要硬件

✅ 不是Tech bubble产品
   - 不是纯给开发者的工具
   - 有真实的非tech用户
```

### 红旗信号（不要做）

```
❌ 需要融资才能做的（marketplace、硬件）
❌ 需要network effect的（社交网络）
❌ 教育类（决策链太长，一人公司做不了）
❌ 需要内容生产的（需要持续投入）
❌ 已经有巨头垄断的（Gmail、Notion）
❌ 完全免费且没有付费计划的
```

## 🎪 实际案例

### 案例1：Time Ledger（时间管理）

```
发现渠道：Product Hunt
信号：
- 238 upvotes，排名#2
- Solo maker from 韩国
- 当前免费，但潜在付费空间

评估：
✅ 需求存在（时间管理是大市场）
❌ 但竞争激烈（Toggl、RescueTime）
❌ 差异化不明显

决定：暂不做
理由：红海市场，需要很强的差异化
```

### 案例2：Brisk Teaching（教育工具）

```
发现渠道：Chrome Extension
信号：
- 500k用户
- 融资$6.9M
- Schools & Districts付费

评估：
❌ 教育市场（决策慢）
❌ 需要销售团队（to B）
❌ 不适合solo

决定：不做
理由：一人公司做不了教育
```

## 💡 寻找的理想产品

### 特征

```
✅ 明确的痛点（能用一句话说清楚）
✅ B2C或小B（个人/小企业付费）
✅ 客单价$5-50/月（不高不低）
✅ 不需要持续内容生产
✅ 技术栈简单（Web App即可）
✅ 竞争对手<10个
✅ 有付费用户（证明willingness to pay）
```

### 好的例子

- AI工具套壳（简单API调用）
- 垂直领域工具（给某个职业的）
- 生产力工具（记录、管理、分析）
- 自动化工具（省时间的）

## 🔄 持续过程

### 每日（10分钟）

- 检查trend-monitor邮件
- 标记2-3个有兴趣的产品

### 每周（30分钟）

- 深入研究5个候选产品
- 填写评估checklist
- 选出1个最值得做的

### 每月（1小时）

- 回顾本月发现的产品
- 分析趋势（什么类型的产品在火）
- 调整筛选标准

## ⚠️ 常见错误

### 错误1：追概念而不是产品

```
❌ "我要做AI产品"
✅ "我要做这个具体的AI图片工具"
```

### 错误2：只看热度不看商业模式

```
❌ "这个产品1000个upvotes！"
✅ "这个产品1000个upvotes，而且有$29/月付费版"
```

### 错误3：纠结太久不动手

```
❌ 研究10个产品，花1个月，还在选
✅ 研究5个产品，花1周，选1个，开始做
```

### 错误4：山寨巨头产品

```
❌ 山寨Notion（太复杂）
✅ 山寨某个Notion的小功能（可行）
```
