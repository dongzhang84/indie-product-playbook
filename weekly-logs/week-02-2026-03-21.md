# Week 2: Build Summary (3/21 – 3/27)

## Project 1: LaunchRadar

- 4th vibe project: pivoted from SaaS to personal tool after competitive research showed the market wasn't ready; open sourced
- **Core feature:** generates relevant keywords + subreddits from a single product description, scans Reddit for real user pain points, and auto-generates AI reply suggestions in sync
- Fixed subreddit recommendation bias, results are now much more targeted
- Removed Stripe and payment walls; simplified to a single-input form to reduce friction
- Someone registered within hours of open sourcing — no promotion at all
- Open source: [github.com/dongzhang84/launchradar](https://github.com/dongzhang84/launchradar)

## Project 2: ProfitPilot (new project — AI profit analyzer for Shopify store owners)

- Kicked off from scratch this week
- Shipped MVP in 3 days
- **Core flow:** Shopify OAuth → data sync → AI profit health check → 30-question AI diagnostic → proactive insights (push, not pull)
- Launched Stripe subscription (Free + Pro $29/mo) with a 7-day free trial
- Redesigned dashboard from "user queries data manually" to "AI tells you what to pay attention to"

## Project 3: Socrates Finds You (personal tool, ongoing)

- Daily runs continuing (days 9–13 this week)

## Project 4: AceRocket (SAT / AP Calculus question bank, on another machine)

- **New vibe workflow experiment:** a single prompt now converts an entire PDF question bank into formatted HTML pages and pushes them to the backend — fully replacing the old manual OCR + question-by-question correction process
- Fully vibe coded the upgrades

## Key Lessons This Week

- **Low competitor traffic doesn't mean no demand** — it means no one has productized the idea yet; personal tool is the fastest validation path
- **Open source is a cold-start channel on its own** — people find you without any promotion
- **The B2B window is closing:** vibe coding is still mostly a developer thing, but by next year agents will be commoditized and competition will be brutal — now is the last window for individuals to build something meaningful with AI
